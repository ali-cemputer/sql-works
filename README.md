# E-Trade Database Analysis & SQL Notes

This repository contains SQL scripts developed in **SSMS** for E-Commerce data analysis, along with my personal study notes.

### Files
* **`Relational_Database_Querys.sql`**: Complex queries for real-world scenarios (Sales, CRM, Shipping).
* **`SQLQueryWorks.sql`**: Advanced educational practice queries.
* **`SQL 101` & `SQL 202`**: Personal study notes and summaries. **(Note: These files are in Turkish)**.

### Key Concepts Demonstrated
* **Joins:** Connecting tables (Orders, Users, Items, Locations).
* **Date Functions:** `DATEDIFF`, `DATENAME` for age and time-based analysis.
* **Aggregations:** `GROUP BY`, `HAVING` for identifying multi-address users and sales trends.
* **Reporting:** Sales analysis by **Hour**, **Day**, and **Region**.

### Database Context
Queries operate on: `Orders`, `OrderDetails`, `Invoices`, `Payments`, `Items`, `Users`, and Location tables.
